# gene-expression-analysis

This is a beginner bioinformatics project using Python.

## 📌 Description
This project demonstrates how to:
- Create a simple gene expression dataset
- Analyze the data using Python
- Calculate summary statistics such as mean and maximum expression levels

## 🛠 Tools Used
- Python
- Pandas

## 🎯 Purpose
I created this project as part of my transition from Biochemistry to Bioinformatics. It reflects my interest in applying data analysis techniques to biological datasets.

## 🚀 Future Improvements
- Load real biological datasets
- Visualize gene expression
- Perform advanced statistical analysis
